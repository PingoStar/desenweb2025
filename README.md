# desenweb2025
Repositório desenvolvimento web 1
1 - para rodar esse projeto você precisa ter um sistema operacional, um web server e um banco de dados.
2 - estruture todo o projeto no VS code.
3 - crie um banco de dados chamado avaliação 
4 - parear a senha do postgres com o projeto para ter o acesso ao banco de dados para regsitros e avaliações
5 - estruturar os departamentos e perguntas de análise necessária no Banco de dados
6 - conetar sempre o xampp para poder visualizar as informações no server
7 - assim que atualizar o projeto enviar para o github.
8 - ter as pastas salvas dentro do web desenweb2025 estruturadas
